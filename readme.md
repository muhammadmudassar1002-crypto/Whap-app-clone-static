# WhatsApp Clone (Static Layout)

This is a static (non-functional) WhatsApp web interface clone built using only HTML and CSS.

Completed as part of **CWI Task 27** from Rehan School's "Coding with AI" series.

## Features
- Left sidebar with list of recent chats
- Right side chat window with message bubbles
- Original WhatsApp green theme and chat background
- Message input box with send button at the bottom
- Fully responsive design (works on mobile & desktop view)
